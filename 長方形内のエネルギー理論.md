# 1. 長方形は「力の場」である

デザイン分野に、単独で確立された「長方形内のエネルギー理論」が存在するわけではない。

実際には、次の考え方を組み合わせて構図を分析する。

* 視覚的重量
* 知覚的な力
* 重心とバランス
* ゲシュタルト心理学
* 図と地
* 視線誘導
* 余白
* 情報密度
* 色彩による強弱
* 映画における運動と時間

長方形は、単に物を置くための箱ではない。

何も置かれていない状態でも、長方形の中には次のような見えない構造が存在する。

```text
┌────────────────────┐
│        上部          │
│  浮遊・上昇・緊張    │
│                    │
│        中心          │
│  安定・集中・停止    │
│                    │
│        下部          │
│  接地・重量・蓄積    │
└────────────────────┘
```

物体は、長方形内のどこに置かれるかによって意味や印象が変わる。

# 2. 中心・上下・左右・角の性質

## 2-1. 中心

中心は、四方の辺までの距離が均等になるため、最も安定しやすい場所である。

### 中心配置が与える印象

* 安定
* 静止
* 完結
* 正面性
* 権威
* 儀式性
* 孤立
* 逃げ場のなさ

中心配置は、エネルギーが少ない状態ではない。

周囲から均等に力を受け、エネルギーが中央に凝縮している状態と考えられる。

映画で人物を真正面の中央に置くと、安心感だけでなく、次の印象も生まれる。

* 監視されている
* 逃げ場がない
* 異様に整いすぎている
* 時間が停止している
* 証明写真のように無機質

> 中央とは、エネルギーが消えている場所ではなく、動かずに凝縮している場所である。

---

## 2-2. 上部

上部にある物体は、地面から離れているため、重力に逆らっているように感じられる。

### 上部が与える印象

* 上昇
* 理想
* 精神性
* 権力
* 解放
* 空
* 浮遊
* 未確定
* 落下の可能性
* 不安定さ

上部の物体は、実際の大きさが小さくても強い緊張を持つことがある。

```text
画面上部の小さな点

        ●
        ↓

落下しそう
浮いている
支えがない
```

### 映画での使用例

人物を上部に小さく配置すると、次のように見える。

* 世界から切り離されている
* 宙に浮いている
* 足場がない
* 超越的
* 不安定

### 名刺での使用例

文字やロゴを上端近くに置くと、次の印象が出る。

* 緊張感
* 細さ
* 高級感
* 冷たさ
* 洗練

ただし端に近すぎると、配置ミスや印刷ズレのようにも見える。

---

## 2-3. 下部

下部は地面や土台を連想させ、物体を支える場所として働く。

### 下部が与える印象

* 接地
* 安定
* 現実
* 重量
* 支持
* 身体性
* 蓄積
* 沈下
* 終了
* 基礎

下に要素を置くと安定しやすいが、情報を詰めすぎると次の印象になる。

* 重苦しい
* 沈んでいる
* 地面に押しつけられている
* 上部が空虚
* 下へ崩れそう

---

## 2-4. 左右

左右の意味は、上下ほど普遍的ではない。

次の要素によって意味が変化する。

* 読字方向
* 文化
* 映像内の移動方向
* 時間の表現
* 画面内の人物関係

左から右へ読む文化では、一般的に次の流れが生まれやすい。

```text
左                          右
過去・入口・既知  →  未来・出口・結果
```

### 左から右への移動

* 前進
* 自然な進行
* 目的地へ向かう
* 物語が進む

### 右から左への移動

* 逆行
* 抵抗
* 帰還
* 過去へ戻る
* 流れに逆らう

ただし、日本語の縦書き、漫画、右開きの本では、右から左が自然に機能する場合もある。

> 左右の意味は自然法則ではなく、文化や文脈による慣習である。

---

## 2-5. 辺

物体が辺に近づくほど、中央から離れるため緊張が増す。

### 辺付近の印象

* 押しつけられている
* 外へ逃げようとしている
* フレームに固定されている
* 画面外へ続いている
* 落ちそう

物体の向きによって意味が変わる。

| 状態       | 印象      |
| -------- | ------- |
| 辺と平行     | 整列、固定   |
| 辺へ向かう    | 脱出、移動   |
| 辺に背を向ける  | 内部へ戻る   |
| 一部が切れている | 画面外への継続 |

---

## 2-6. 角

角は、上下と左右の二つの辺が交差する場所である。

そのため、辺よりも圧力が強い。

### 角付近の印象

* 追い詰められている
* 閉じ込められている
* 外へ逃げようとしている
* 装飾として固定されている
* 強く孤立している

右上に小さなロゴを置いただけでも目立つことがある。

これは次の要因が重なるためである。

* 中心から遠い
* 角に近い
* 周囲から孤立している
* コントラストが高い

---

# 3. 視覚的重量

視覚的重量とは、物体の実際の重さではなく、

> その要素が、どれだけ強く目を引きつけるか

を表す考え方である。

## 視覚的に重く感じる要素

| 要因   | 重くなりやすい状態    |
| ---- | ------------ |
| 大きさ  | 大きい          |
| 明暗差  | 背景との差が大きい    |
| 彩度   | 鮮やか          |
| 輝度   | 極端に明るい、または暗い |
| 輪郭   | 硬く鮮明         |
| 情報量  | 細部が多い        |
| 孤立   | 周囲から離れている    |
| 位置   | 中心から遠い、上部、角  |
| 形    | 複雑、不規則       |
| 向き   | 斜め           |
| 意味   | 顔、目、文字、手、危険物 |
| 奥行き  | 手前に見える       |
| 動き   | 動いている、動きそう   |
| 規則違反 | 周囲の並びから外れている |

## 意味による重量

顔、目、手、文字などは、小さくても強い視覚的重量を持つ。

例えば、大きな灰色の丸より、小さな赤い目の方が目立つことがある。

```text
大きな薄い円  ≒  小さな赤い点
```

重要なのは面積だけではない。

```text
視覚的重量
＝ 面積
＋ コントラスト
＋ 色
＋ 位置
＋ 意味
＋ 孤立
```

# 4. 位置エネルギーと視覚的モーメント

中心から離れるほど、物体には視覚的な緊張が生まれる。

これは、中心につながれたゴムを外側へ引っ張る状態に近い。

```text
中央                          辺
安定       →       緊張・離脱

        ●────────→
```

## 視覚的モーメント

構図を分析するときは、物理の「てこ」のように考えると分かりやすい。

```text
視覚的モーメント
＝ 視覚的重量 × 基準点からの距離
```

これは科学的に厳密な計算式ではなく、構図を考えるための比喩である。

### 例

```text
左側：大きな灰色の人物
右側：小さな赤い文字
```

右側の文字が次の条件を持つ場合、大きな人物と釣り合うことがある。

* 赤い
* 高コントラスト
* 右端に近い
* 周囲から孤立している
* 読むべき意味を持つ

したがって、左右の面積を同じにする必要はない。

# 5. 視線・動き・線が生むベクトル

画面内のエネルギーには、重量だけでなく方向がある。

この方向性を、ベクトルとして考える。

## ベクトルを生む要素

* 人物の視線
* 顔の向き
* 指差し
* 歩く方向
* 車の向き
* 道路
* 遠近法の消失線
* 建物の稜線
* 光
* 影
* モーションブラー
* 文字列
* 三角形
* 斜線

---

## 視線の先にある余白

人物が見ている方向には、目に見えない力が発生する。

### 視線方向に余白がない場合

```text
┌────────────────┐
│             顔→│
└────────────────┘
```

視線がフレームに衝突するため、次の印象になる。

* 圧迫
* 行き止まり
* 不安
* 追い詰められている
* 見たくない物が画面外にある

### 視線方向に余白がある場合

```text
┌────────────────┐
│     顔→          │
└────────────────┘
```

次の印象になる。

* 期待
* 思考
* 会話相手の存在
* 未来
* 移動可能性

この余白は空白ではない。

人物の視線によって、意味や緊張を与えられた空間である。

---

# 6. 余白のエネルギー

余白は、何も存在しない空間ではない。

要素同士の関係や、視線の流れを決める重要な構造である。

## 余白の三つの役割

### 1. 分離する

要素同士を離し、別のグループとして見せる。

### 2. 強調する

周囲に何もない要素は孤立し、強く目立つ。

### 3. 方向を作る

物体の片側に広い余白があると、その方向へ動く可能性が生まれる。

```text
●
              広い余白
```

上の点は、右側へ進もうとしているように感じられる。

---

## ゲシュタルト心理学との関係

人間は、個別の要素を完全に独立して見るのではなく、まとまりとして認識する。

主な法則は次のとおり。

| 法則   | 内容                 |
| ---- | ------------------ |
| 近接   | 近い物は同じグループに見える     |
| 類似   | 色や形が似た物は同じ仲間に見える   |
| 連続   | 線や流れが自然につながって見える   |
| 閉合   | 欠けていても一つの形として補完する  |
| 共通領域 | 同じ枠内の物は一つのグループに見える |

余白は、

> 何と何が関係し、何と何が別物なのか

を決定する。

# 7. 違和感の正体

違和感は、単純に「エネルギーが多い」から生まれるとは限らない。

より正確には、

> エネルギーの交通整理に失敗している状態

である。

## 7-1. 焦点が多すぎる

次の要素がすべて同じ強さで主張している状態。

* 大きな顔
* 赤い文字
* 明るいロゴ
* 派手な背景
* 太い装飾
* 強い光

見る側は、最初にどこを見ればよいか決められない。

---

## 7-2. 主役以外が一番重い

名刺で名前を読ませたいのに、装飾の赤線が最初に見える場合など。

これは情報量の問題ではなく、

> 階層が逆転している

ことが原因である。

---

## 7-3. ベクトルが分散している

```text
人物の視線   →
背景の線     ←
文字の傾き   ↗
光の方向     ↓
```

すべての力が別方向へ向かうと、画面が裂けるように感じられる。

---

## 7-4. 画面外へ出る力が強すぎる

人物、視線、道、光などが、すべて外側へ向かっていると、鑑賞者の目も画面外へ追い出される。

### 映画では有効な場合

* 別離
* 不安
* 逃走
* 孤独
* 画面外の脅威

### 名刺や広告では問題になりやすい

* 名前から目が離れる
* 情報を読まずに視線が抜ける
* ロゴが弱くなる

---

## 7-5. 密度差に理由がない

片側だけ細部が密集し、反対側が空いている場合でも、意味やバランスがあれば成立する。

しかし、理由なく偏っていると次の印象になる。

* 配置ミス
* 未完成
* 片側だけ重い
* 余白が余った

---

## 7-6. 接線が発生している

接線とは、二つの要素が偶然ぎりぎりで触れている状態である。

### 例

* 人物の頭頂と背景の線が接する
* 文字の端と図形の端が接する
* ロゴと罫線がほぼ触れている
* 人物の指とフレーム端が接する

重なっているのか、離れているのかが曖昧になり、小さな衝突が生まれる。

---

## 7-7. 規則と例外が中途半端

整列した要素の中で、一つだけずれると非常に目立つ。

意図的ならアクセントになる。

意図がなければ、単なるミスに見える。

```text
●  ●  ●  ●
```

```text
●  ●   ●  ●
```

下の並びでは、中央のずれが強いエネルギーを持つ。

---

## 違和感の基本式

```text
違和感
＝ エネルギーの総量
```

ではない。

```text
違和感
＝ 意図した情報階層
－ 実際に目を引く順序
```

と考えると分かりやすい。

# 8. 対称・非対称とバランス

## 8-1. 対称構図

```text
        │
   ●    │    ●
        │
```

### 対称が与える印象

* 安定
* 権威
* 正式
* 神聖
* 完成
* 停止
* 制度
* 秩序

完全な対称は、安心感だけでなく次の印象にもなる。

* 無機質
* 不気味
* 支配的
* 時間が止まっている
* 逃げ場がない

---

## 8-2. 非対称構図

```text
●                     ○ ○
```

左側の大きな物体を、右側の複数の小さな物体で支える構図。

左右の形が同じでなくても、視覚的重量が釣り合えば安定する。

### 非対称が与える印象

* 動き
* 現代性
* 自然さ
* 会話
* 発展
* 緊張
* 変化

---

## 8-3. 動的平衡

バランスが取れていることと、静止していることは同じではない。

斜線や視線を使えば、

> 崩れないが、動こうとしている状態

を作れる。

これを動的平衡として考える。

---

# 9. 線と形が持つエネルギー

## 9-1. 水平線

水平線は地面や水平面を連想させる。

### 印象

* 安定
* 休息
* 静けさ
* 広がり
* 距離
* 停止
* 死

---

## 9-2. 垂直線

垂直線は、重力に逆らって立つ形である。

### 印象

* 人間
* 建築
* 意志
* 成長
* 権威
* 緊張
* 高さ

---

## 9-3. 斜線

斜線は水平にも垂直にも安定していない。

### 印象

* 運動
* 転倒
* 加速
* 衝突
* 不安
* 攻撃
* 変化

```text
┌────────────────┐
│          ／     │
│       ／        │
│    ／           │
└────────────────┘
```

長い斜線は、画面全体を切り分けるため非常に強い。

---

## 9-4. 三角形

三角形は、頂点と底辺を持つため方向性や階層を作る。

```text
        頂点
         ▲
        / \
       /   \
      ─────
        基礎
```

| 向き  | 印象        |
| --- | --------- |
| 上向き | 上昇、権威、集中  |
| 下向き | 落下、危険、不安定 |
| 横向き | 移動、攻撃、矢印  |

人物を三角形状に配置すると、複数人でも一つのグループとして見えやすい。

---

## 9-5. 円

円には明確な始点と終点がない。

### 印象

* 完結
* 柔らかさ
* 循環
* 保護
* 中心性
* 親密さ
* 永続性

ただし文脈によっては、次のようにも見える。

* 穴
* 目
* 標的
* 監視
* 閉鎖

---

## 9-6. 四角形

四角形はフレームそのものと似た構造を持つ。

### 印象

* 安定
* 規則
* 人工物
* 制度
* 固定
* 閉鎖
* 管理

長方形内に別の長方形を置くと、フレーム内フレームになる。

人物をその中へ置くことで、次の表現ができる。

* 拘束
* 監視
* 注目
* 分離
* 閉じ込め

---

# 10. 色・明暗・図と地

## 10-1. 図と地

人間は画面を、次の二つに分けて認識する。

* **図**：見る対象
* **地**：背景

### 図と地が明確な場合

* 読みやすい
* 主役が分かる
* 安定する
* 商業デザインに向く

### 図と地が曖昧な場合

* 不気味
* 夢のよう
* 主体が不明確
* 二重意味
* 迷い
* 存在感の消失

映画では、人物と背景の明度を近づけることで、人物を闇へ溶け込ませられる。

名刺では、文字と背景の明度差が小さいと単に読みにくくなる。

---

## 10-2. 色の重量

色の力は、色名だけでは決まらない。

次の要素によって変わる。

* 彩度
* 明度
* 面積
* 背景との差
* 周囲の色
* 文脈
* 使用される場所

### 高彩度色

* 前へ出る
* 活動的
* 人工的
* 強調
* 警告

### 低彩度色

* 後退する
* 静か
* 古い
* 自然
* 上品

### 暖色と寒色

一般的には暖色が前へ出て、寒色が後退しやすい。

ただし、

* 鮮やかな青
* 濁った赤

を比べた場合、青の方が強く見えることもある。

> 色の強さは、色そのものより周囲との差で決まる。

---

## 10-3. 小さな色の焦点

画面全体が青灰色で、小さな黄色が一つだけある場合、黄色が第一焦点になる。

```text
色の重量
＝ 色そのもの
ではなく
＝ 周囲との違い
```

---

# 11. 視覚的エントロピー

視覚的エントロピーとは、画面内の秩序と乱雑さの程度を考える方法である。

## 低エントロピー

### 特徴

* 対称
* 反復
* 少ない色
* 一定の間隔
* 単純な形
* 明確な階層

### 印象

* 安定
* 清潔
* 高級
* 制度的
* 予測可能
* 静的

---

## 高エントロピー

### 特徴

* 不規則
* 多色
* 密度差
* 複雑な形
* 重なり
* 複数方向
* 予測不能

### 印象

* 活気
* 混乱
* 自然
* 危険
* 若さ
* 熱狂
* 不安定

---

## 情報量と混乱は別

```text
情報量が多い
≠
混乱
```

```text
情報量が整理されていない
＝
混乱
```

要素が多くても、次の条件があれば成立する。

* 主役が明確
* 線が同じ焦点へ集まる
* 色の役割が決まっている
* 大きな形から小さな形へ読める
* 反復にリズムがある

反対に、要素が二つしかなくても、両方が同じ強さで争っていれば落ち着かない。

---

## 秩序と乱れ

優れた構図では、

> 理解できる秩序の中に、少量の予測不能性がある

ことが多い。

```text
秩序だけ
→ 安定するが退屈

乱れだけ
→ 刺激はあるが理解できない

秩序＋少量の乱れ
→ 興味と緊張が生まれる
```

---

# 12. 長方形の比率による違い

## 12-1. 横長

横長画面は、左右方向への動きが強くなる。

### 印象

* 映画的
* 風景
* 会話
* 移動
* 時間の流れ
* 左右の対立

人物の左右に空間を作りやすいため、視線や移動を表現しやすい。

---

## 12-2. 縦長

縦長画面は、上下の差が強調される。

### 印象

* 人物
* 階層
* 上昇
* 落下
* 建築
* 全身
* 孤立

縦動画では頭上と足元の余白に大きな意味が生まれる。

---

## 12-3. 正方形

正方形は中心への吸引力が強い。

### 印象

* アイコン
* 正面性
* 完結
* 静止
* 商品
* 象徴

中心配置と相性がよいが、横方向の物語や移動は作りにくい。

---

## 12-4. 極端に細長い長方形

長辺方向への流れが強くなる。

### 印象

* 帯
* 時間軸
* 道
* スクロール
* 連続
* 圧縮
* 進行方向

---

# 13. 映画におけるエネルギー

映画では、構図のエネルギーが時間とともに変化する。

静止画だけでなく、前後のカットとの関係が重要になる。

## 映画特有のエネルギー

### 1. 画面内運動

人物や物がフレーム内で動く。

### 2. カメラ運動

* パン
* ティルト
* ドリー
* トラック
* ズーム
* 手持ち
* 回転

### 3. カットによる移動

視線の焦点が、別の位置へ瞬間的に移る。

### 4. フレーム外の力

* 音
* 視線
* 影
* 風
* 光
* 声

によって画面外の存在を感じさせる。

### 5. 時間的蓄積

静かな構図が続いた後に斜めの構図を出すと、単体以上に強い衝撃が生まれる。

---

## カット間のエネルギー

映画編集では、一枚ずつの美しさだけでは不十分である。

```text
前のカットの焦点
        ↓
次のカットのどこへ着地するか
```

を考える。

### 自然につながる例

```text
カットA：人物が右を見る
カットB：右側に対象を配置する
```

視線が自然に移動する。

### 意図的に衝突させる例

```text
カットA：人物が右を見る
カットB：左側に対象を配置する
```

次の印象を作れる。

* 驚き
* 空間の断絶
* 混乱
* 方向転換

---

## 映画での画面外空間

人物が画面外を見ると、見えていない空間にもエネルギーが生まれる。

### 使用例

* 誰かがいる
* 脅威が近づいている
* 逃げ道がある
* 観客だけが見られない
* 存在しない物を見ている

画面外は、何も存在しない空間ではなく、音や視線によって意味を与えられる。

---

# 14. 画像・ポスター・サムネイル

静止画では、すべての情報が同時に表示される。

そのため、見る順番を設計する必要がある。

```text
第1焦点
  ↓
第2焦点
  ↓
第3焦点
  ↓
全体理解
```

## 映画ポスターの例

1. 俳優の顔
2. タイトル
3. 背景の状況
4. 公開日
5. 補足情報

すべてを第一焦点にすると、画面が騒がしくなる。

---

## サムネイル

サムネイルは小さく表示されるため、細部のエネルギーが失われる。

### 必要な要素

* 大きな形
* 強い明暗差
* 少ない文字
* 明確な顔や物体
* 単純な図と地
* 一つの主役

細かい装飾だけでバランスを取っても、縮小時には機能しない。

---

# 15. 名刺におけるエネルギー

名刺は、映画やポスターよりも低エネルギーかつ高精度である必要がある。

目的は鑑賞ではなく、情報を正しく伝えることだからである。

## 名刺の主な目的

* 名前を認識する
* 所属を理解する
* 肩書を確認する
* 連絡先を読む
* 人柄や企業の印象を感じる

---

## 基本的な情報階層

```text
1. 名前
2. 所属・肩書
3. 電話・メール・住所
4. ロゴ・装飾
```

個人用の場合は、次の構成も考えられる。

```text
1. 名前
2. 地域・住所
3. 電話番号
4. 装飾
```

---

## 名刺の余白

余白は次の印象を作る。

* 丁寧
* 信頼
* 落ち着き
* 高級
* 清潔
* 情報が整理されている

ただし、余白を広くすれば必ず高級になるわけではない。

名前が小さすぎたり、端へ寄りすぎたりすると次の印象になる。

* 弱い
* 不安
* 未完成
* 配置ミス
* 情報が足りない

---

## 名刺における上下構造

上部に装飾、中央に名前、下部に連絡先を置く場合は、次の構造になる。

```text
上部：装飾
      ↓
導入・象徴

中央：名前
      ↓
主役・中心

下部：連絡先
      ↓
基礎・接地
```

上下の役割が明確になるため、自然に読みやすくなる。

---

## 落ち着いた名刺を作る方法

* 高彩度色を小面積にする
* 太い罫線を避ける
* 中央揃えか穏やかな左揃えにする
* 名前と連絡先の強弱を明確にする
* 装飾を角へ押しつけすぎない
* 上部を軽くする
* 下部に情報の基礎を作る
* 余白に意味を持たせる
* ロゴを名前より強くしすぎない

---

# 16. 実制作での確認方法

## 16-1. ぼかして見る

画面を強くぼかし、細部を消す。

確認する内容は次のとおり。

* 一番暗い場所
* 一番明るい場所
* 大きな塊
* 全体の重心
* 最初に目が行く場所

ぼかしても主役が分かれば、大きな構造は整理されている。

---

## 16-2. 白黒で見る

色を除去し、明暗だけで情報階層が成立しているか確認する。

色を消した途端に主役が分からなくなる場合、色だけに依存している。

---

## 16-3. 左右反転する

長時間見続けると、配置のズレに慣れてしまう。

左右反転すると次の問題を見つけやすい。

* 重心の偏り
* 顔の傾き
* 余白の不自然さ
* 意図しない斜線
* 左右の不均衡

---

## 16-4. 縮小する

サムネイル程度まで縮小する。

確認する内容は次のとおり。

* 最初に何が見えるか
* 文字が読めるか
* 主役が残っているか
* 装飾だけが目立っていないか

---

## 16-5. 目を細める

細部を見えにくくし、面積とコントラストだけを見る。

ぼかし表示が使えない場合にも有効。

---

## 16-6. 矢印を書き込む

画面上へ、すべての方向性を書き込む。

```text
顔の向き   →
道の方向   ↗
光の方向   ↓
文字の流れ →
```

矢印がどこへ集まるか、どこから外へ出るかを確認する。

---

## 16-7. 要素を一つずつ隠す

装飾、背景、線、ロゴなどを一つずつ非表示にする。

### 確認する質問

* なくても成立するか
* なくした方が主役が強くならないか
* その要素は何を支えているか
* その要素が第一焦点になっていないか
* 作品の意味と一致しているか

---

## 16-8. 第一焦点を確認する

画面を一秒だけ見て、最初に見えたものを確認する。

その要素が制作者の意図した主役と違う場合、情報階層を修正する必要がある。

---

# 17. 実用的なエネルギー式

以下は科学的な測定式ではなく、制作判断のためのモデルである。

## 視覚的重量

```text
視覚的重量 W
＝ 大きさ
＋ コントラスト
＋ 彩度
＋ 鮮明さ
＋ 情報量
＋ 意味の強さ
＋ 孤立度
```

---

## 位置的緊張

```text
位置的緊張 T
＝ 視覚的重量 W
× 中心または安定領域からの距離
```

---

## 画面内エネルギー

```text
画面内エネルギー
＝ 視覚的重量
＋ 位置的緊張
＋ 視線・線・動きのベクトル
＋ フレームとの衝突
＋ 色や明暗の差
```

---

## 違和感

```text
違和感
＝ エネルギーの総量
```

ではない。

```text
違和感
＝ 意図した階層と
   実際に目を引く順番のずれ
```

---

## 構図の基本関係

```text
安定だけ
→ 退屈

緊張だけ
→ 混乱

安定＋緊張
→ 構図
```

---

# 18. 結論

良い構図とは、すべてを均等に配置することではない。

視覚的エネルギーを扱うとは、次のことを設計することである。

* どこに重量を置くか
* どこへ視線を動かすか
* どこで視線を止めるか
* どこを空けるか
* どこに緊張を作るか
* どこで規則を破るか
* 何を最初に見せるか

違和感の多くは、要素が多いこと自体から生まれるのではない。

> 画面が発している力と、作品が伝えたい意味が一致していないこと

から生まれる。

同じ人物でも、配置によって意味が変わる。

### 中央対称に置く場合

* 静かな受容
* 権威
* 停止
* 儀式
* 孤立

### 上端や角へ置く場合

* 不安
* 追い詰められている
* 足場がない
* 逃げようとしている
* 画面外へ消えそう

物体そのものだけが意味を作るのではない。

> 長方形のどこに置かれ、周囲の余白、辺、色、線、視線から、どのような力を受けているか

によって、構図の意味と印象が決まる。
